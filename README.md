# schema-forge-site

Official website for [Schema Forge](https://github.com/xubylele/schema-forge).

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Lint & format

```bash
npm run lint
npm run format
npm run format:check
```

## Preview deployment

Connect this repository to [Vercel](https://vercel.com) for automatic preview deployments on pull requests and production deploys on merge to `main`.
